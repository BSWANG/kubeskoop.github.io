# 云供应商

KubeSkoop支持的云供应商如下：

## 阿里云

指定`--cloud-provider aliyun`以使用阿里云作为云提供商。

| 参数 | 说明 |
| --- | --- |
| `--aliyun-access-key-id`        | 阿里云access key                                                                       |
| `--aliyun-access-key-secret`    | 阿里云access secret                                                                    |
| `--aliyun-security-token`       | (可选)阿里云security token                                                              |
